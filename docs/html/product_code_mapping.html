Mapped Database Tables

products Write
Holds core item details, localized titles/descriptions (English & Hindi), price, ratings, and active status.
stock_history Write
Logs delta changes in stock numbers, including the change reasons (e.g. Sales, Restock, Admin Edit).
buy_requests Read
Stores purchase requests for out-of-stock items, used to bypass stock checks on restocking.

Relationships & Cascades

C
Deleting a product triggers ON DELETE CASCADE for: product_images, product_variants, reviews, stock_history, and buy_requests.
S
It triggers SET NULL on: categories, order_items.
