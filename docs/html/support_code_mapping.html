Mapped Database Tables

support_messages Write
Stores initial queries received from customers, including contact information and open/resolved status indicators.
support_replies Write
Stores communication history (replies) linked to support messages. Cascade deleted if the main ticket is cleared.
faqs Read
Stores frequently asked questions and answers served on the customer help center.
