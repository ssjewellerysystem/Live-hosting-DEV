Mapped Database Tables

users Write
Stores profile data, login info, and status indicators. Triggers cascade deletes on sessions.
delivery_addresses Write
Linked one-to-one to user. Handles street address, city, state, and pin code. Deleted if user profile is dropped.
carts Write
Assigned one-to-one to user to manage active cart items. Created on registration.
